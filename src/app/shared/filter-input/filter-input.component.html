Aqui você pode escrever os trechos a serem considerados. Pode colocar palavras completas ou apenas trechos.


Não fará diferença se digitar minúsculas ou maiúsculas, mas você deve usar acentos corretamente.


Caso queria especificar uma expresão, coloque entre aspas. Ex: ["na escola" sucesso] (o filtro considerará 2 e não 3 expressões.)


Para números, também é possível expecificar um intervalo usando hífen: [2018-2024]


Por fim, escolha se é necessário que os resultados contenham todas as expressões (Combinação com 'E') ou apenas uma (Combinação com 'Ou').
